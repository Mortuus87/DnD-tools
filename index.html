Pen & Paper Tools

Welcome to this site. It contains several useful tools for anyone running tabletop roleplaying games, like Dungeons & Dragons, or Pathfinder, with more to come in the future.

Features to be implemented:
  • City/Town stat gen. Click here for a peak at the WIP
  • DC table
  • Dice roller?
  • All-in-one generator. NPCs with ability scores, character traits, and any other related stats
Features thast may be implemented if server side support is added:
  • NPC database. Save a stat block
  • NPC generation using abilities and trait generator
Spellbook Generator

A generator for wizard/magus spellbooks. Other caster spell lists are available, but may not work properly. Besides, you should plan out a sorcerer better than with a generator!
